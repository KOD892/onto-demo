# Ont ID Login Guide

This project lets you log in using your **ONTO Wallet Extension** and **Ont ID**.

## Steps to Run the Project

1. **Clone the Repository**

   ```bash
   git clone https://github.com/barnazaka/ont-demo
   cd Ont-demo
   ```

2. **Install Dependencies**

   ```bash
   npm install
   ```

3. **Run the Project**

   ```bash
   node server.js
   ```
